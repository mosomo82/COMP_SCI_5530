# METADATA FOR Telco_Customer_Churn.csv

## 1. DATASET INFORMATION

Description: This dataset contains information about a fictional telecommunications company. Each row represents a customer, and the columns contain customer attributes. The "Churn" column indicates whether the customer left within the last month. The dataset is often used for predicting customer churn. 

Rows: 7,043

Columns: 21

---

## 2. SOURCE INFORMATION

* **Original File Name:** Telco_Customer_Churn.csv
* **Source URL:** https://www.kaggle.com/datasets/mosapabdelghany/telcom-customer-churn-dataset
* **Format:** Comma-Separated Values (CSV)

---

## 3. COLUMN DESCRIPTIONS

This dataset contains 21 columns: 


1. **customerID**	
   * **Description:** The unique ID of each customer.
   * **Data Type:** Text (Alphanumeric)
   * **Values:** e.g., "7590-VHVEG", "5575-GNVDE"

2. **gender**
   * **Description:** The customer's gender.
   * **Data Type:** Text (Categorical)
   * **Values:** "Female", "Male"

3. ** SeniorCitizen**
   * **Description:** Indicates if the customer is a senior citizen.
   * **Data Type:** Numeric (Categorical)
   * **Values:** 1 (Yes), 0 (No)

4. **Partner**
   * **Description:** Indicates if the customer has a partner.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No"

5. **Dependents**
   * **Description:** Indicates if the customer has dependents.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No"

6. **tenure**
   * **Description:** The number of months the customer has been with the company.
   * **Data Type:** Numeric (Integer)
   * **Values:** 1 - 72

7. **PhoneService**
   * **Description:** Indicates if the customer has phone service.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No"

8. **MultipleLines**
   * **Description:** Indicates if the customer has multiple phone lines.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No", "No phone service"

9. **InternetService**
   * **Description:** The type of internet service the customer has.
   * **Data Type:** Text (Categorical)
   * **Values:** "DSL", "Fiber optic", "No"

10.**OnlineSecurity**
   * **Description:** Indicates if the customer has online security service.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No", "No internet service"

11.**OnlineBackup**
   * **Description:** Indicates if the customer has online backup service.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No", "No internet service"

12.**DeviceProtection**
   * **Description:** Indicates if the customer has device protection service.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No", "No internet service"

13.**TechSupport**
   * **Description:** Indicates if the customer has tech support service.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No", "No internet service"

14.**StreamingTV**
   * **Description:** Indicates if the customer has a TV streaming service.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No", "No internet service"

15.**StreamingMovies**
   * **Description:** Indicates if the customer has a movie streaming service.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No", "No internet service"

16.**Contract**
   * **Description:** The customer's contract term.
   * **Data Type:** Text (Categorical)
   * **Values:** "Month-to-month", "One year", "Two year"

17.**PaperlessBilling**
   * **Description:** Indicates if the customer uses paperless billing.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No"

18.**PaymentMethod**
   * **Description:** The customer's payment method.
   * **Data Type:** Text (Categorical)
   * **Values:** "Electronic check", "Mailed check", "Bank transfer (automatic)", "Credit card (automatic)"

19.**MonthlyCharges**
   * **Description:** The amount charged to the customer monthly.
   * **Data Type:** Numeric (Continuous)
   * **Values:** e.g., 29.85, 56.95

20.**TotalCharges**
   * **Description:** The total amount charged to the customer.
   * **Data Type:** Numeric (Continuous)
   * **Values:** e.g., 29.85, 1889.5

21.**Churn**
   * **Description:** The target variable, indicating if the customer has churned.
   * **Data Type:** Text (Categorical)
   * **Values:** "Yes", "No"
